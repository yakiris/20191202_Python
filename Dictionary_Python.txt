-----> Простые типы данных
int - целое число
float - число с плавающей точкой
bool - логический тип данных True/False
None - ничего (пустой тип)
str - строка
-----> Доп.математические операции
// - целая часть от деления
% - остаток от деления
** - возведение в степень
-----> Команды
print('Привет, Python!') - вывод разных типов данных через , (выводится через пробел)
   print('Привет, Python!', sep=';') - разные разделители слов
   print('Привет, Python!', end='|') - разные разделители строк
input('Введите данные') - ввод данных
-----> Условные операторы
if условие:- оператор условия
elif другое условие: - оператор условия
else: - если условия не выполнились
