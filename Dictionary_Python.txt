-----> Простые типы данных
int - целое число
float - число с плавающей точкой
bool - логический тип данных True/False
None - ничего (пустой тип)
str - строка. состоит из набора символов

-----> Доп.математические операции
// - целая часть от деления
% - остаток от деления
** - возведение в степень
-----> Команды
print('Привет, Python!') - вывод разных типов данных через , (выводится через пробел)
   print('Привет, Python!', sep=';') - разные разделители слов
   print('Привет, Python!', end='|') - разные разделители строк
input('Введите данные') - ввод данных
-----> Условные операторы
if условие:- оператор условия
elif другое условие: - оператор условия
else: - если условия не выполнились
while: - цикл с условием. Выполнется пока условие == True
break - выход из цикла
continue - переход на следующую итерацию цила (команды после continue не выполняются)
while: else: - если условия цикла не выполнились, то цикл выходит в else
str[1] -  обращение к символу слова по индексу с начала слова. индексы начинаются с 0
str[-1] -  обращение к символу слова по индексу с конца слова. индексы начинаются с 0
str[1:4] - срез символов от начального индекса символа 1 до конечного 4
str[:4] - срез символов сначала строки до конечного 4
str[1:] - срез символов от начального индекса символа 1 до конца строки
-----> Основные функции и методы для работы со строками
len(str) - функция показывает количество символов в строке
str.finde('а') - метод ищет символ а в строке. вовзвращает индекс найденого символа, если смвола с троке нет возвращает -1
str.split() - разбивает строку через пробел, (;) - через любой другой символ
str.isdigit() - проверяет, что строка состоит только из чисел
str.upper() - приведение строки к верхнему регистру
str.lower() - приведение строки к нижнему регистру
format - форматирование строки. 'Привет (). Тебе () лет?'.format('Ivan', 30)
list - списки - упорядоченные изменяемые коллекции произвольных типов. list = ['первый', 'второй', 'третий']. 
   В списках как и в строке работают индексы и срезы, которые возвращают элемент.
-----> Основные функции и методы для работы со списками
len(list) - функция показывает количество элементов в списке
list.upper('четвертый') - метод добавляет элемент в конец спискам
list.pop() - удаляет последний элеент и возвращает его
list.clear() - очищает весь список
liat.remove('четвертый') - удаление элемента из списка по значению
del list[0] - удаление элемента из списка по его индексу
